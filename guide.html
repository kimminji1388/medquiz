새 HTML 추가

  3. 화면 아래의 Commit changes를 누릅니다.
incoming에 HTML 올리기

같은 파일명은 기존 시험을 수정하고, 다른 파일명은 새 시험으로 추가합니다.

2

자동 변환 확인

  1. Actions 화면에서 Import quiz HTML을 확인합니다.
  2. 노란색은 작업 중, 초록색은 완료, 빨간색은 실패입니다.
  3. 초록색이 되면 문제풀이 사이트를 새로고침합니다.
Actions 확인 사이트 새로 열기

3

지난 시험 보관

  1. 문제 묶음 관리 화면을 엽니다.
  2. 숨길 시험의 스위치를 끕니다.
  3. Download settings를 눌러 question-sets.json을 받습니다.
  4. 아래 설정 업로드 버튼을 눌러 받은 파일을 올립니다.
  5. Commit changes를 누른 뒤 사이트를 새로고침합니다.
문제 묶음 관리 설정 파일 업로드

보관한 문제의 데이터와 풀이 기록은 삭제되지 않습니다. 스위치를 다시 켜면 복구할 수 있습니다.
